**"mi portfolio"**

Mi trabajo consiste en una página web sobre mi currículum o portfolio.

En ella hablo sobre mí, mi educación desde la primaria hasta la actualidad, menciono mis hobbies, experiencias laborales, habilidades y mis trabajos independientes. Luego muestro algunos de mis trabajos más recientes a través de imágenes.

También se pueden encontrar mis redes sociales para contactarme; las mismas tienen vínculos directos para facilitar la tarea del cliente.

Todo mi trabajo fue realizado en Visual Studio Code, utilizando HTML, CSS y Flexbox. Me queda pendiente utilizar herramientas de Figma, ya que al intentar utilizarlas tuve algunas trabas.

La medida de pantalla utilizada fue de 1280 x 800.
